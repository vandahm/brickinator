What is this?

This project helps Peter and his associates design Lego mosaic art for purposes that only Peter truly understands. If you are one of those associates, contact Peter to get access.

Is this like Lego Art Remix?

Yes, and it uses a lot of the same code. The difference is that this project lets you save in-progress work and choose the size of the squares that get assembled into the final mosaic. Also, it makes prettier PDF instructions.

Lego Art Remix GitHub
I want to download this.

You probably don't. You almost certainly want to use the original Lego Art Remix, which runs entirely in your browser and doesn't require a server-side backend. That said, the project is on GitHub if you really need it.
